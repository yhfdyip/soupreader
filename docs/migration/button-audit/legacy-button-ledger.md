# Legacy 按钮总台账（T02）

- 任务ID：`T02`
- 来源目录：`../legado/app/src/main/res/menu/*.xml`
- 统计基线：`90` 个菜单文件，`410` 个 `<item>` 条目
- 生成时间：`2026-02-20`

## 提取口径

- 仅统计菜单 XML 中 `<item>` 节点（含嵌套 `<menu>` 内条目）。
- 分组字段 `group_path` 记录当前所在 `<group android:id>` 栈路径；无分组记为 `-`。
- 本台账用于后续 `T03/T04` 建立 soupreader 入口台账与一对一映射。

## 可复现命令

```bash
perl docs/migration/button-audit/extract_legacy_menu_items.pl > /tmp/legacy_buttons.tsv
find ../legado/app/src/main/res/menu -maxdepth 1 -name '*.xml' | wc -l
xmllint --xpath 'count(//item)' ../legado/app/src/main/res/menu/*.xml
```
